# **Projeto de Autenticação com Streamlit e PostgreSQL** 🔐

Este projeto é uma aplicação de autentic1ação simples e segura, desenvolvida com **Python**, **Streamlit** e **PostgreSQL**. Ele permite que os usuários realizem login e cadastro, com validação robusta de entradas e hashing seguro de senhas usando **bcrypt**.

## **Funcionalidades** 📋

- **Login:**

  - Entrada com email e senha.
  - Validação de credenciais com hashing seguro.

- **Cadastro:**

  - Validação de campos obrigatórios.
  - Prevenção de emails duplicados.
  - Hashing seguro de senhas antes de salvar no banco de dados.
  - Mensagens de feedback para o usuário em caso de erro ou sucesso.

- **Validação de Inputs:**

  - Email válido (formato correto).
  - Senhas confirmadas e seguras.

## **Tecnologias Utilizadas** 🛠

- **Python 3.9+**
- **Streamlit** – Framework para criação de interfaces interativas.
- **PostgreSQL** – Banco de dados relacional.
- **psycopg2** – Biblioteca para integração com PostgreSQL.
- **bcrypt** – Hashing de senhas.
- **dotenv** – Para gerenciamento seguro de variáveis de ambiente.
